# Usage

- [x] Deploy
    1. install go & set gopath
    2. clone this repo & enter the project folder
    3. run ```chmod +x ./deploy.sh & ./deploy.sh```

- [ ] Create project
    1. mkdir for your project & cd to the folder
    2. run ```solpack -c```

- [x] Pack project
    1. enter the project folder
    2. run ```solpack -b```

- [x] Documentation
    - run ```solpack -h for help```
